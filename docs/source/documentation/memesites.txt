MemeSite
==============

MemeSite is the superclass that defines the interface for all the meme-extraction subclasses.
The subclasses should implement caching to improve reading performance.

   :members:

Origins Enum
=============

This is an enum-type designed to provide abstraction for the origins of the memes we extract from the Internet.
